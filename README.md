# Rabbit Game

There are 100 blocks in the game. At the start of each round, the Rabbit spawns at a random position between 1 and 100.

The player has 50 bullets. Your goal is to shoot the Rabbit before you run out of bullets.

Game rules:

* After every missed shot, the Rabbit randomly moves one block to the left or right.
* If the Rabbit reaches block 1 or block 100, it instantly moves to block 50.
* You win if you successfully shoot the Rabbit.
* You lose if all 50 bullets are used before hitting the Rabbit.
